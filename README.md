## 项目概述 python409

本项目是一款基于Python语言的招聘数据分析系统，集成了招聘信息的爬取、存储、可视化展示与数据挖掘功能。系统采用轻量级的Web应用框架Flask，前端框架Bootstrap，数据库为Mysql，数据可视化部分使用Echarts进行展示。

## 技术栈

- 后端：Python
- Web框架：Flask
- 前端框架：Bootstrap
- 数据库：Mysql
- 数据可视化：Echarts

## 功能模块

- **招聘信息爬取**：自动获取网络上的招聘信息数据。
- **数据存储**：将爬取的数据保存在Mysql数据库中。
- **数据可视化**：通过Echarts展示数据的统计与分析结果。
- **数据挖掘**：包括薪资预测与岗位匹配功能。

## 系统角色

- **用户**：使用系统进行招聘信息的查询与分析。
- **管理员**：负责系统的维护与数据管理。

## 运行环境

- 操作系统：兼容Windows、Linux、MacOS等。
- Python版本：3.6及以上。
- 需安装的依赖：Flask、Bootstrap、Mysql、Echarts。

## 部署步骤

1. 安装Python环境及相应的依赖包。
2. 配置Mysql数据库并创建相应数据库与表。
3. 运行系统后端服务。
4. 通过浏览器访问前端页面。

